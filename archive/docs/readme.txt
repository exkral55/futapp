# FUTAPP ver1 – Understat Top5 Extractor

Bu klasör, projenin ilk çalışan sürümüdür.
Understat üzerinden 5 büyük lig + çoklu sezon oyuncu sezon istatistiklerini çekmek için kullanılmıştır.

Dosya:
main_ver1_understat.py

Çalıştırma:
python archive/main_ver1_understat.py

Çıktılar:
archive/output/*.csv

Bu sürüm arşivlenmiştir. Yeni geliştirme FUTAPP ana main.py üzerinden devam eder.
